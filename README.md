# Makeup Website – HTML & CSS Project

A static makeup-themed website designed using **HTML and CSS**.
The website showcases beauty and makeup content with a visually appealing layout and styled components.

## 💄 Overview

This project is a front-end website created to practice **web design and layout techniques** using HTML and CSS. The website presents makeup-related content in a structured and attractive format.

The goal of the project was to build a responsive and visually engaging webpage while improving front-end development skills.

## 🚀 Features

* Modern and clean website design
* Styled layout using CSS
* Organized webpage sections
* Visual presentation of makeup products or beauty content
* Static website that can run in any browser

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
